NFC Message Beam APIs

Create a dialog which can send the message to NFC Beam Slave

Returns:QPython NFC json result
AndroidHelper.NFCBeamMessage(message)¶

Sendthe message to NFC Beam Slave without dialog

Returns:QPython NFC json result
AndroidHelper.dialogCreateNFCBeamSlave()¶

androidhelper.getLastKnownLocation()¶

location = location.get('network', location.get('gps'))
